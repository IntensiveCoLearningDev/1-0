# 完整测试 1.0

## 共学简介

你是否有过运动的冲动，但总是坚持不到第三周？是否收藏了一堆健身视频，却从没点开第二次？我们不是训练你成为运动员，而是帮你把运动变成像刷牙一样自然的习惯。

**适合谁？**

* 曾经坚持运动过一段时间，但中断后难以重启的人

- 想建立长期运动习惯、而非追求短期效果的人

* 愿意每天花30-60分钟，通过实战巩固运动行为模式

**参与后你能获得什么？**

* 一套可复用的习惯养成方法（基于《微习惯》《原子习惯》等经典）

- 8周循序渐进的运动计划，从低强度开始，逐步提升耐力和多样性

* 每天打卡监督与同伴激励，告别独自挣扎

- 结营时一份个人运动成长记录与公开分享展示

## 共学内容 / 课程信息

### 第1周：启动与基线测试

* **主题**：建立最小运动单元，找到当前体能基线

- **学习材料**：

* **[NHS 运动指南](https://www.nhs.uk/live-well/exercise/)**（了解不同强度标准）

- **[《微习惯》第1-3章](https://www.amazon.com/Micro-habits-Stephen-Guise/dp/1891219)**（理解“小到不可能失败”的策略）

* **练习任务**：每天选择一种运动（快走、瑜伽、自重训练等），每次至少15分钟，记录运动类型和时长

- **交付物**：每日运动截图（如运动app记录、手写日志照片）+ 简单文字感受

* **反馈方式**：微信群内互相点评打卡，助教每周总结

### 第2周：增加时长与多样性

* **主题**：将单次运动延长至30分钟，尝试至少2种不同运动

- **学习材料**：

* **[ACSM 运动测试与处方指南（摘要）](https://www.acsm.org/education-resources/books/acsm-recommendations-on-exercise)**（安全原则）

- **[FitOn App（免费家庭训练）](https://fitonapp.com/)**（推荐几类课程）

* **练习任务**：每天运动30分钟，每周至少3天更换运动类型

- **交付物**：每周运动多样性报告（列出不同运动类型及时长）

### 第3周：加入强度变化

* **主题**：引入中高强度间歇，提升心肺能力

- **学习材料**：

* **[Tabata 训练介绍](https://www.tabatatraining.com/)**（或类似认证网站）

- YouTube 频道 **[FitnessBlender](https://www.youtube.com/user/FitnessBlender)**（筛选20分钟HIIT视频）

* **练习任务**：每周2天进行20分钟HIIT，其余保持30分钟中等强度

- **交付物**：每周心率数据（若无可简单描述呼吸急促程度）

### 第4周：力量与核心稳定

* **主题**：加入简单的力量训练，预防运动损伤

- **学习材料**：

* **[NHS 力量训练指南](https://www.nhs.uk/live-well/exercise/strength-exercises/)**（7个基础动作）

- **[《徒手力量训练》视频](https://www.youtube.com/playlist?list=PLoGsyqHqLdq7fVvz3BxhHx8Q5b7W)**（选择5个动作）

* **练习任务**：每次运动包含10分钟力量训练（俯卧撑、深蹲、弓步等），每周至少2天

- **交付物**：记录力量动作的组数、次数，拍摄关键动作照片
